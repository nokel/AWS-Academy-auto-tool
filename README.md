# AWS-Academy-auto-tool
A userscript tool that makes aws academy a little less annoying

 - automatically turn off subtitles, 
 - automatically go to next video in module, and
 - automatically play video in next module.

Should also work when you go directly into video and play from where you left off, but not 100% sure.

How to use:

Download "Violent Monkey" https://github.com/violentmonkey/violentmonkey (Or should also work with tampermonkey)

after addon is installed click "add script"

Copy and paste the script from this github into the userscript agent

make sure the script is enabled and looking for awsacademy.instructure.com/* and/or emergingtalent.contentcontroller.com/*
